Geosocial

A real-time geolocation social app built with Next.js and Leaflet 1.9.4.
Each user is represented on the map with a marker, allowing you to visualize active users and their locations in an interactive, browser-based interface.

This project is built as a Firebase Studio Next.js starter, making it easy to integrate with Firebase authentication, Firestore, or real-time database features.
