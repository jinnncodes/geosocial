# **App Name**: Geo Social

## Core Features:

- Interactive Map: Display a map interface using Leaflet 2.0, showing user locations.
- User Location Pins: Pin user locations on the map based on data from `dummyusers.ts`.
- Dynamic Location Updates: Real-time updates to user location pins on the map, reflecting changes in `dummyusers.ts`.
- User Selection: Ability to select different users from `dummyusers.ts` and focus the map on their location.

## Style Guidelines:

- Primary color: Sky blue (#87CEEB) for the map interface, evoking a sense of exploration and openness.
- Background color: Light gray (#F0F0F0), providing a neutral backdrop that doesn't distract from the map.
- Accent color: Bright orange (#FFA500) for user location markers, ensuring they stand out on the map.
- Font: 'Inter', a sans-serif font, for a modern and clean interface. Use 'Inter' for both headlines and body.
- Use simple, clear icons for user selection and map controls.
- Maintain a clean and intuitive layout with the map taking center stage and user controls easily accessible.
- Use smooth transitions when updating user locations on the map for a seamless user experience.
